# nvim-mjml supports

### Components
1. `mjml`
2. `mj-head`
3. `mj-include`

### Standard Head Component
1. `mj-attributes`
2. `mj-breakpoint`
3. `mj-font`
4. `mj-html-attributes`
5. `mj-preview`
6. `mj-style`
7. `mj-title`

### Standard Body Component
1. `mj-accordion`
2. `mj-button`
3. `mj-carousel`
4. `mj-column`
5. `mj-divider`
6. `mj-group`
7. `mj-hero`
8. `mj-image`
9. `mj-navbar`
10. `mj-raw`
11. `mj-section`
12. `mj-social`
13. `mj-spacer`
14. `mj-table`
15. `mj-text`
16. `mj-wrapper`
17. `Ending tags` e.g mj-text or mj-buttons
